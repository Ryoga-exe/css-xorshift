About this

Xorshift implemented in 100% CSS using the CSS trick called The CPU Hack. (no JavaScript!)

A "CPU Hack" implies unlocking the ability for continuous crunching of data and re-evaluation of state.

Note: This has been confirmed to work with Google Chrome version 119.0.6045.160 and Microsoft Edge 119.0.2151.72. It may not function correctly with other browsers.
